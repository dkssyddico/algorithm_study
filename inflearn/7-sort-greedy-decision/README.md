이중 for문으로 한다.
i는 인덱스 0번부터 끝까지 돈다.
j는 i 뒷편부터 돈다.
i번째에 있는 아이템에 올 제일 작은 숫자를 0번부터 끝까지 돌아 찾아서 i번째에 갖다놓는다.
idx 변수에 제일 작은 숫자를 지정. 0부터 시작한다.
arr[j]값과 arr[idx]값을 비교한 뒤 arr[j] 값이 더 작으면 idx를 j로 바꾼다.
그리고 arr[i]와 arr[idx] 를 배열에서 순서를 바꾸는데,
arr[idx]를 왼쪽으로 갈 수 있도록 순서를 바꾼다.
