# 프로그래머스 정렬 문제

## 1. H-index (21.09.07)

위키백과 설명을 보면,
어떤 연구자의 h 지수는 그 사람이 쓴 모든 논문 중 n회 이상 인용된 논문이 n개 이상일 때, 이 둘을 동시에 만족하는 n의 최대값으로 한다.

배열을 내림차순으로 정렬한다. 논문 인용횟수를 한차례씩 돌면서 n회 이상 인용된 논문이 n개보다 같거나 클 때를 구해준다.
